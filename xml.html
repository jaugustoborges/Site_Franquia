Manual XML

Este manual destina-se às empresas contratantes do MID-e Central. MID-e é uma aplicação Middleware utilizada para comunicação dos sistemas Linx com a Secretaria da Fazenda dos Estados para emissão de Nota Fiscal Eletrônica (NF-e) e Nota Fiscal de Consumidor Eletrônica (NFC-e).

Para acessar o portal é necessario os seguintes programas instalados no computador:

Silverlight https://www.microsoft.com/silverlight
Internet Explorer https://www.microsoft.com/pt-br/download/internet-explorer.aspx

Acesso ao Portal

Para acessar o portal do MID-e, o usuário Cliente deverá acessar o Link do endereço: http://portal.mide.linxsaas.com.br/portal. O sistema irá apresentar a tela Login:

Informe usuário e senha e clique no botão Entrar. Será apresentada a tela inicial.
Caso o usuário Cliente esteja realizando o seu primeiro acesso, o sistema irá solicitar que seja cadastrada uma nova senha.

Informe a senha atual e a nova senha. Em seguida reinforme a nova senha e clique no botão OK.
A nova senha de acesso que acaba de ser cadastrada será salva e será apresentada a tela inicial do sistema.

Download XML's

Acesso ao Módulo: MID-e Cliente Master / Movimentações / Download de XMLs

O sistema irá apresentar a tela Download de XMLs.
Nesta tela o usuário poderá realizar o download dos arquivos (em lote ou individualmente).

No agrupamento Download de XML.
No campoGrupo Econômico clique na Lupa.
No campo Cadastro clique na 1ª Lupa para selecionar o ID do cadastro e na 2ª Lupa para selecionar o nome do emissor.
No campo CNPJ clique na Lupa para selecionar o CNPJ do emissor.
No campo Emissão informe um range de datas para a pesquisa do período (00/00/0000 a 00/00/0000).
Selecione um dos checkbox NF-e, NFC-e/CFe SAT, NF-e de Entrada.
O sistema irá preencher os campos Grupo Econômico,Cadastro eCNPJ conforme o Grupo Econômico selecionado na guia Preferências no Menu Grupo Econômico vinculado ao usuário logado.
Os outros campos (Id. Nota, Nr. Nota Inicial, Nr. Nota Final e Status Nota) são opcionais para a pesquisa.
Clique no botão Pesquisar Informações.
O Sistema deverá realizar a busca e apresentar a relação das notas na gridNotas da telaDownload de XMLs composta pelas seguintes colunas (Selecionar, Id. Produto, Produto, Id. Cadastro, Razão Social, Id Nota, Nr. Nota Inicial, Nr. Nota Final, Série Nf e Emissão ).

Clique no botão Marcar Todas da grid do Agrupamento Notas.
O sistema irá selecionar o checkbox da coluna Selecionar Selecionar para todas as notas.
Clique no botão Download de XMLs da grid do Agrupamento Notas.
Informe o diretório onde o arquivo *.zip será armazenado na máquina.
Informe o nome do arquivo e local onde o mesmo será salvo.
Em seguida o sistema apresentara a pop-up de aviso: "Download realizado com sucesso! Botão OK".
Clique no botão OK da pop-up de aviso. O sistema irá fechar a pop-up.
No local informado deverá conter o arquivo gerado no formato XML.

Quando a consulta ultrapassar os 1000 documentos ou ocorrer algum problema de lentidão ou timeout, o sistema irá apresentar a pop-up de informação abaixo:

Clicando em Sim, o sistema solicita informar um endereço de e-mail, digite e clique em Agendar.
